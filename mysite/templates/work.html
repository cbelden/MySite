MTS Systems Corporation

Automated UI Testing: C#

This past summer Calvin interned as a software test engineer at MTS Systems in Eden Prairie, MN. MTS produces test devices for a variety of fields including the materials, automotive, and aerospace industries, and MTS provides a rich Windows application, TestSuite, that controls several of these testing rigs.

Cal was a member the automated test team that worked with TestSuite, and he spent the summer implementing UI drivers and writing automated UI test cases in C#. It was rewarding converting tedious manual tests (that usually get squeezed into the end of a release) into automated processes that could be run every night.

Gentex Corporation

Firmware Dev: C

Integration Testing: Ruby

The summer of 2012 Calvin interned as a software engineer at Gentex in Zeeland, Michigan. Gentex manufactures car mirrors, and Cal wrote commands to retrieve and send diagnostic information to and from the mirror.

Calvin also wrote integration tests to ensure the car mirror's core features were behaving appropriately. He tested features like automatic dimming and displaying accurate compass and GPS information.
