MTS Systems Corporation

Automated UI Testing: C#

This past summer I interned as a software test engineer at MTS Systems in Eden Prairie, MN. MTS produces testing devices for a variety of fields, and a rich Windows application, TestSuite, controls several of these devices.

I was a member of the automated test team that worked with TestSuite, and I focused on developing the coded UI test framework. The framework will drive nightly-run acceptance tests that will help minimize the risk of expensive bugs.

I also participated in the annual Hackathon and built a POC, web-based, intelligent phone service for customer feedback submissions. The service allowed customers to query the status of existing feedback items, create new items, and receive updates: all over the phone.

Gentex Corporation

Firmware Dev: C

Integration Testing: Ruby

The summer of 2012 I interned as a software engineer at Gentex, a global car mirror manufacturer in Zeeland, Michigan. I wrote diagnostic commands to communicate and send commands to the mirror.

In addition, I also wrote integration tests to ensure the car mirror's core features were behaving appropriately. I tested features like automatic dimming and displaying accurate compass and GPS information.
