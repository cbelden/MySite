The Tools

This site was built using the Python web framework, Flask. Flask is a lightweight framework built to get web apps up and running quickly.

All of the pages on this site are rendered using Jinja2 templates and the Twitter Boostrap toolkit, which have both been incredibly easy to use! This site is hosted on Heroku using their free tier (which has been great!).
