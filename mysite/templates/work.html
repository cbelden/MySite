MTS Systems Corporation

Automated UI Testing: C#

This past summer I interned as a software test engineer at MTS Systems in Eden Prairie, MN. MTS produces test devices for a variety of fields including the materials, automotive, and aerospace industries. MTS provides a rich Windows application, TestSuite, that controls several of these testing rigs.

I was a member the automated test team that worked with TestSuite, and I spent the summer implementing UI drivers and writing UI test cases in C#. It was rewarding converting tedious manual tests into automated processes that could be run every night (vs. at the end of every release!).

Gentex Corporation

Firmware Dev: C

Integration Testing: Ruby

The summer of 2012 I interned as a software engineer at Gentex in Zeeland, Michigan. Gentex manufactures car mirrors, and I wrote commands to retrieve and send diagnostic information to and from the mirror.

One common task was retrieving and writing data stored in the mirror's EEPROM. I also wrote integration tests to ensure that the mirror was performing appropriately (correctly dimming, displaying accurate compass readings, displaying accurate GPS information, etc.).
