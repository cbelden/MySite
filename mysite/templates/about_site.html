The Tools

This site was built using Flask. Flask is a lightweight framework built to get Python web apps up and running quickly.

All of the pages on this site are rendered using Jinja2 templates and the Twitter Boostrap toolkit, which have both been really easy to use. This site is hosted on Heroku using their free tier (which has been great!).
