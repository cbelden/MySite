The Tools

This site was built using the Python web framework, Flask. Flask is a lightweight framework built to get web apps up and running quickly. It's great for a simple expose, like this one, but I've been told that it can be difficult (but not impossible) to scale for production purposes.

All of the pages found here are rendered using Jinja2 templates, which I have found to be incredibly easy to use. This site is hosted on Heroku using their free tier (which has been great!).
